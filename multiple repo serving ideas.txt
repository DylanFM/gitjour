$ gitjour
Serve up and use git repositories via Bonjour/DNSSD.

Usage: gitjour <command> [args]

  list
      Lists available repositories.

  serve-dir [<path_to_directory_with_repositories>]
      Serve all the git repostories from the current directory or the specified path
  
  serve
      Serve all the git repositories configured in `git config --global --get gitjour.repopaths`

  add [<path_to_git_repository>]
      Add the current or specified git repository to the config
  
  rm <path_to_git_repository>
      Remove the specified git repository from config


# Concept for gitjour serving API

$ gitjour serve
You haven't added any repositories.

Add a repository with:
  gitjour add ~/mygitrepo

or serve everything in a directory with:
  gitjour serve-dir ~/repos

$ gitjour add ~/mygitrepo
mygitrepo added

$ gitjour serve
Serving ~/mygitrepo

^C$ cd ~/repos
$ ls
repo1 repo2
$ gitjour serve-dir # Current dir
Serving ~/repos/repo1
Serving ~/repos/repo2

^C$ cd .
$ ls
$ gitjour serve-dir ~/repos # Specify the dir
Serving ~/repos/repo1
Serving ~/repos/repo2
